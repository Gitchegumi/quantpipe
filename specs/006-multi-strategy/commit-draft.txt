## Architecture Highlights

- **State Isolation**: Each strategy maintains independent balance/position state
- **Atomic Aggregation**: Weighted PnL combines results without cross-contamination
- **Risk Cascade**: Per-strategy limits checked before portfolio-level global abort
- **Determinism**: Stable run IDs enable exact result reproduction
- **Validation Early**: Unknown strategies rejected before execution starts
- **Weights Tolerance**: Sum validation within 1e-6 epsilon for floating precision

## CLI Examples

```powershell
# Register strategies
poetry run python -m src.cli.run_backtest --register-strategy alpha --strategy-module my_strategies.alpha

# List registered strategies
poetry run python -m src.cli.run_backtest --list-strategies

# Run multi-strategy with custom weights
poetry run python -m src.cli.run_backtest `
  --data price_data/processed/eurusd/test.csv `
  --strategies alpha beta gamma `
  --weights 0.5 0.3 0.2

# Equal-weight fallback
poetry run python -m src.cli.run_backtest `
  --data price_data/processed/eurusd/test.csv `
  --strategies alpha beta

# Run with progress bar (default for CLI)
poetry run python -m src.cli.run_backtest `
  --data price_data/processed/eurusd/test.csv `
  --direction BOTH
# Output: Ingesting 6,922,364 candles ------------------------------ 100% • 327142 gaps
```
